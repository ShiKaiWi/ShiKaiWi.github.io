异步编程

异步编程是 Javascript 的一大特色，因为 Javascript 是单线程工作，因此如果没有异步方式的话，基本上是用户体验将会是无法描述。
因此 Javascript 的语法对异步的看重使得其对异步编程用了比较大的力气去优化，本文将会对直到 ES7 的异步编程方式进行总结：

  1. Callback
  2. Promise
  3. Generator
  4. asyn + await

但是在阐述这些方式之前，有一个概念是需要事先阐述的是 Javascript 的事件处理机制。
